# NLP Foundations: Text Analytics

## Module Overview
This module introduces Natural Language Processing (NLP) through practical text
analytics tasks using transparent and interpretable techniques.

---

## Intended Learning Outcomes
Students will learn to:
1. Pre-process textual data
2. Convert text into numerical representations
3. Build simple text classification models
4. Understand limitations and bias in NLP systems

---

## Topics Covered
- Text cleaning and tokenisation
- TF-IDF representations
- Basic text classification
- Practical use cases (feedback, surveys, logs)

---

## Pedagogical Emphasis
- Conceptual clarity over technical complexity
- Strong links to real-world applications
- Awareness of ethical and contextual risks

---

## Assessment Ideas
- Text analytics mini-project
- Error analysis exercise
- Discussion on responsible NLP usage
